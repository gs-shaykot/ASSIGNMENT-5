Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a speical feature of javascript that represents the structure of a webpage as a tree of objects. It allows scripts to access, modify, and interact with the content, structure, and style of HTML documents.

How do you select an element from the DOM?

Using getElementById('elementId'), getElementsByClassName('className')
getElementsByTagName('tagName')
querySelector('selector') or
querySelectorAll('selector')

What is event delegation in the context of the DOM, and why is it useful?

It is a technique of the DOM, where we attach only one event listener to a parent element rather than attaching listeners to multiple individual child elements.

usefulness: Event delegation is useful because it improves efficiency by reducing the number of event listeners and enhances maintainability by simplifying the management of event listeners in large HTML documents.

How do you manipulate an element's attributes and styles using the DOM?

We can use DOM Methods like: getAttribute(attributeName), removeAttribute(attributeName), setAttribute(attributeName, value) to manipulate an element's attributes
